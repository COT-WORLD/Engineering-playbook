# macOS OS Shortcuts

## Finder / File Management

- `Cmd + N` → New Finder window
- `Cmd + Shift + N` → New folder
- `Cmd + O` → Open selected item
- `Cmd + Delete` → Move selected item to Trash
- `Cmd + Shift + Delete` → Empty Trash
- `Cmd + D` → Duplicate selected item
- `Cmd + I` → Get Info on selected item
- `Cmd + Option + I` → Show Inspector
- `Cmd + Shift + G` → Go to folder
- `Cmd + L` → Make alias (shortcut) of selected item

## Window Management

- `Cmd + M` → Minimize window
- `Cmd + Option + M` → Minimize all windows
- `Cmd + H` → Hide current app
- `Cmd + Option + H` → Hide other apps
- `Cmd + Tab` → Switch apps
- `Cmd + ~` → Switch windows of same app
- `Ctrl + Up Arrow` → Mission Control
- `Ctrl + Down Arrow` → App Exposé
- `Cmd + Space` → Spotlight search
- `Ctrl + Left / Right Arrow` → Switch desktops

## Screenshots

- `Cmd + Shift + 3` → Full screen screenshot
- `Cmd + Shift + 4` → Select area screenshot
- `Cmd + Shift + 4 + Space` → Screenshot a window
- `Cmd + Shift + 5` → Screenshot / screen recording menu

## Text Editing

- `Cmd + C / Cmd + V` → Copy / Paste
- `Cmd + X` → Cut
- `Cmd + Z / Cmd + Shift + Z` → Undo / Redo
- `Option + Left / Right Arrow` → Move cursor by word
- `Cmd + Up / Down Arrow` → Go to beginning / end of document
- `Cmd + A` → Select all
- `Cmd + F` → Find
- `Cmd + G / Shift + Cmd + G` → Find next / previous

## Power User Tips

- `Cmd + Option + Esc` → Force Quit apps
- `Cmd + Option + Power` → Sleep display
- `Cmd + Option + D` → Show / hide Dock
- `Fn + Delete` → Forward delete
- `Option + Cmd + V` → Paste and match style
- `Cmd + Control + Q` → Lock screen
