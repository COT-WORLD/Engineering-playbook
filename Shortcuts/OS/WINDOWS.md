# Windows OS Shortcuts

## File Explorer

- `Win + E` → Open File Explorer
- `Alt + D` → Focus address bar
- `Ctrl + Shift + N` → New folder
- `F2` → Rename selected file/folder
- `Ctrl + C / Ctrl + V` → Copy / Paste
- `Ctrl + X` → Cut

## Window Management

- `Win + Up` → Maximize window
- `Win + Down` → Minimize window / restore
- `Win + Left / Right` → Snap window to side
- `Win + Shift + Left / Right` → Move window to another monitor
- `Alt + Tab` → Switch apps
- `Ctrl + Shift + Esc` → Task Manager

## Virtual Desktops

- `Win + Ctrl + D` → New virtual desktop
- `Win + Ctrl + Left / Right` → Switch desktop
- `Win + Ctrl + F4` → Close current desktop

## General Shortcuts

- `Win + L` → Lock screen
- `Win + S` → Search
- `Win + I` → Settings
- `Win + R` → Run dialog
- `Ctrl + Z / Ctrl + Y` → Undo / Redo

## Power User Tips

- `Shift + Right Click` → Show extra context menu options
- `Ctrl + Shift + Enter` → Run as administrator
- `Win + .` → Emoji panel
