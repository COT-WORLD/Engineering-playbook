# Linux OS Shortcuts

## General GUI Shortcuts (Ubuntu / GNOME)

- `Ctrl + Alt + T` → Open terminal
- `Alt + Tab` → Switch apps
- `Alt + Esc` → Switch windows in order
- `Super + Up / Down / Left / Right` → Maximize / Minimize / Snap windows
- `Super + D` → Show desktop
- `Super + L` → Lock screen
- `Ctrl + Alt + L` → Lock screen (alternative)
- `Print Screen` → Screenshot entire screen
- `Shift + Print Screen` → Screenshot selected area
- `Alt + Print Screen` → Screenshot current window

## File Manager (Nautilus / Files)

- `Ctrl + N` → New window
- `Ctrl + H` → Show hidden files
- `Ctrl + T` → New tab
- `F2` → Rename file
- `Ctrl + C / Ctrl + V` → Copy / Paste
- `Ctrl + X` → Cut
- `Ctrl + A` → Select all

## Terminal Shortcuts (Bash / Zsh)

- `Ctrl + A` → Move cursor to beginning of line
- `Ctrl + E` → Move cursor to end of line
- `Ctrl + U` → Delete to beginning of line
- `Ctrl + K` → Delete to end of line
- `Ctrl + W` → Delete word before cursor
- `Ctrl + Y` → Paste last deleted text
- `Ctrl + R` → Reverse search command history
- `!!` → Run last command
- `!n` → Run nth command from history
- `Tab` → Auto-complete file names / commands

## Virtual Desktops

- `Ctrl + Alt + Arrow Left / Right` → Switch desktops
- `Ctrl + Alt + Shift + Arrow` → Move window to another desktop

## Power User Tips

- `Ctrl + Shift + T` → Open new terminal tab
- `Ctrl + Shift + C / Ctrl + Shift + V` → Copy / Paste in terminal
- `xkill` → Click a window to force close
- `htop` → Interactive process manager
- `sudo !!` → Re-run last command as root
- `alias name='command'` → Create terminal alias for repetitive commands
