# VSCode Shortcuts - Windows

## General

- `Ctrl + Shift + P` → Open Command Palette
- `Ctrl + P` → Quick Open (go to file)
- `Ctrl + Shift + N` → New window
- `Ctrl + Shift + W` → Close window
- `Ctrl + ,` → Open settings
- `F1` → Command Palette (alternative)
- `Ctrl + K Ctrl + S` → Keyboard shortcuts reference

## File Management

- `Ctrl + N` → New file
- `Ctrl + O` → Open file
- `Ctrl + S` → Save
- `Ctrl + Shift + S` → Save as
- `Ctrl + K S` → Save all
- `Ctrl + F4` → Close editor
- `Ctrl + W` → Close tab

## Editing

- `Ctrl + C / Ctrl + V / Ctrl + X` → Copy / Paste / Cut
- `Ctrl + Z / Ctrl + Y` → Undo / Redo
- `Ctrl + /` → Toggle comment
- `Shift + Alt + A` → Toggle block comment
- `Ctrl + D` → Select next occurrence
- `Ctrl + Shift + L` → Select all occurrences
- `Alt + Up / Down` → Move line up / down
- `Shift + Alt + Up / Down` → Copy line up / down
- `Ctrl + Shift + K` → Delete line
- `Ctrl + Enter` → Insert line below
- `Ctrl + Shift + Enter` → Insert line above

## Multi-Cursor

- `Alt + Click` → Insert cursor
- `Ctrl + Alt + Down / Up` → Insert cursor below / above
- `Ctrl + Shift + L` → Select all occurrences of current selection
- `Ctrl + F2` → Select all occurrences in document

## Navigation

- `Ctrl + T` → Go to symbol
- `F12` → Go to definition
- `Alt + F12` → Peek definition
- `Ctrl + Shift + O` → Go to symbol in file
- `Ctrl + G` → Go to line
- `Ctrl + - / Ctrl + Shift + -` → Navigate back / forward

## Search & Replace

- `Ctrl + F` → Search in file
- `Ctrl + H` → Replace in file
- `F3 / Shift + F3` → Next / previous match
- `Ctrl + Shift + F` → Search across files
- `Ctrl + Shift + H` → Replace across files

## Debugging

- `F9` → Toggle breakpoint
- `F5` → Start / continue debugging
- `Shift + F5` → Stop debugging
- `F11 / Shift + F11` → Step into / Step out
- `F10` → Step over

## Terminal

- `Ctrl + `` → Toggle terminal
- `Ctrl + Shift + `` → Create new terminal
- `Ctrl + C` → Copy in terminal
- `Ctrl + V` → Paste in terminal
- `Ctrl + ↑ / ↓` → Scroll terminal history
