# VSCode Shortcuts - macOS

## General

- `Cmd + Shift + P` → Open Command Palette
- `Cmd + P` → Quick Open (go to file)
- `Cmd + Shift + N` → New window
- `Cmd + Shift + W` → Close window
- `Cmd + ,` → Open settings
- `F1` → Command Palette (alternative)
- `Cmd + K Cmd + S` → Keyboard shortcuts reference

## File Management

- `Cmd + N` → New file
- `Cmd + O` → Open file
- `Cmd + S` → Save
- `Cmd + Shift + S` → Save as
- `Cmd + K S` → Save all
- `Cmd + W` → Close tab
- `Cmd + Option + W` → Close all tabs

## Editing

- `Cmd + C / Cmd + V / Cmd + X` → Copy / Paste / Cut
- `Cmd + Z / Cmd + Shift + Z` → Undo / Redo
- `Cmd + /` → Toggle comment
- `Shift + Option + A` → Toggle block comment
- `Cmd + D` → Select next occurrence
- `Cmd + Shift + L` → Select all occurrences
- `Option + Up / Down` → Move line up / down
- `Shift + Option + Up / Down` → Copy line up / down
- `Cmd + Shift + K` → Delete line
- `Cmd + Enter` → Insert line below
- `Cmd + Shift + Enter` → Insert line above

## Multi-Cursor

- `Option + Click` → Insert cursor
- `Cmd + Option + Down / Up` → Insert cursor below / above
- `Cmd + Shift + L` → Select all occurrences of current selection
- `Cmd + F2` → Select all occurrences in document

## Navigation

- `Cmd + T` → Go to symbol
- `F12` → Go to definition
- `Option + F12` → Peek definition
- `Cmd + Shift + O` → Go to symbol in file
- `Cmd + G` → Go to line
- `Cmd + - / Cmd + Shift + -` → Navigate back / forward

## Search & Replace

- `Cmd + F` → Search in file
- `Cmd + H` → Replace in file
- `F3 / Shift + F3` → Next / previous match
- `Cmd + Shift + F` → Search across files
- `Cmd + Shift + H` → Replace across files

## Debugging

- `F9` → Toggle breakpoint
- `F5` → Start / continue debugging
- `Shift + F5` → Stop debugging
- `F11 / Shift + F11` → Step into / Step out
- `F10` → Step over

## Terminal

- `Ctrl + `` → Toggle terminal
- `Cmd + Shift + `` → Create new terminal
- `Cmd + C` → Copy in terminal
- `Cmd + V` → Paste in terminal
- `Cmd + ↑ / ↓` → Scroll terminal history
